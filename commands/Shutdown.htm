Shutdown

Kann das System herunterfahren, neustarten oder abmelden.

Shutdown, Code

Parameter

Code

Eine Kombination von den unten genannten Codes.

Bemerkungen

Der Code ist eine Kombination von folgenden Werten:

Abmelden 0
Herunterfahren 1
Neustarten 2
Erzwingen 4
Ausschalten 8
Anhalten/Ruhezustand Siehe DllCall-Beispiel im untersten Abschnitt.
Bildschirm ausschalten Siehe PostMessage-Beispiele.

Addieren Sie die gewünschten Werte. Zum Beispiel ist zum Herunterfahren und Ausschalten des Systems eine 9 notwendig (Herunterfahren + Ausschalten = 1 + 8 = 9). Alternativ kann ein Ausdruck wie 1+8 angegeben werden.

Der "Erzwingen"-Wert (4) zwingt alle offenen Programme sich zu schließen. Diese Methode sollte nur im Notfall verwendet werden, weil offene Programme dadurch Daten verlieren könnten.

Der "Ausschalten"-Wert fährt das System herunter und schaltet den Strom aus.

Des Weiteren kann mithilfe von OnExit() oder OnExit ein Skript dazu gebracht werden, zu erkennen, wann das System herunterfährt oder der Benutzer sich abmeldet.

Siehe auch

Run, ExitApp, OnExit(), OnExit

Beispiele

Erzwingt einen Neustart (Neustart + Erzwingen = 2 + 4 = 6).

Shutdown, 6

Ruft die API-Funktion "SetSuspendState" von Windows auf, um das System anzuhalten oder in den Ruhemodus zu versetzen.

; Parameter #1: Übergeben Sie 1 statt 0, um Ruhezustand statt Anhalten zu bewirken.
; Parameter #2: Übergeben Sie 1 statt 0, um Anhalten ohne Nachfrage zu erzwingen.
; Parameter #3: Übergeben Sie 1 statt 0, um alle Aufwachereignisse zu deaktivieren.
DllCall("PowrProf\SetSuspendState", "int", 0, "int", 0, "int", 0)
